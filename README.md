<h1 align=center>Discovery and Lost</h1>
<p align=center>딥러닝을 이용한 분실물 관리 시스템</p>
<br>

<br>

## Overview
D&L is a system that helps people find lost items. People can easily register lost items through deep running.
<br><br>

## Feature
<ul>
  <li>Regist Lost Item
  <li>Find Lost Item
  <li>Buy Lost Item
</ul>
<br><br>
